Sử dụng Visual Studio để chạy project
Để chỉnh cấu hình từ **x64** sang **x86** trong Visual Studio, bạn làm theo các bước sau:

### Cách đổi cấu hình sang x86:

1. Ở thanh trên cùng của Visual Studio, nhìn phía gần nút "Local Windows Debugger" bạn sẽ thấy ô đang ghi là `x64` hoặc `x86` 
2. Nhấn vào **mũi tên xuống** bên cạnh `x64` hoặc `x86` đó.
3. Trong danh sách xổ xuống, chọn **`x86`**.
